Расчет эквивалетного сопротивления¶

В схеме, приведённой на рисунке:

• R – эквивалентное сопротивление цепи, значение которого определяется при статистическом моделировании,

• R1 находится в диапазоне R1= 30 ± 10 % c равномерным распределением

• R2- это нормальная случайная величина со средним = 30 и СКО = 5

• R3 находится в диапазоне R3 = 60 ± 20 % c равномерным распределением Объем выборки n = 1500.

По результатам статистического моделирования необходимо выполнить следующее:

  1. Вычислить и построить отъюстированные гистограммы для исходных переменных R1, R2, R3 и результирующей переменной R.
  2. Вычислить вероятность P(R) попадания результирующей переменной R в интервал (R- R R+), где R - = МИН ( МЕДИАНА R,СРЕДНЕЕ R ) – СКО R, R + = МАКС ( МЕДИАНА R,СРЕДНЕЕ R) + СКО R

Rmin = min (np.mean(R), np.median(R)) - np.std(R)
Rmax = max (np.mean(R), np.median(R)) + np.std(R)

kmin=0
kmax=0

for i in range(N-1):
    if R[i] < Rmin:
        kmin += 1;
    if R[i] > Rmax:
        kmax += 1;

P = (2000-kmin-kmax)/2000

print ('R- = ', Rmin)
print ('R+ = ', Rmax)
print ('kmin = ', kmin)
print ('kmax = ', kmax)
print('P = ', P)
